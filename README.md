# Personal Portfolio Website 🌐✨

Welcome to my personal portfolio repository! This is a clean, modern, and fully responsive portfolio website designed and developed from scratch to showcase my skills and projects in frontend web development.

## 🚀 Live Demo
You can view the live website here: **https://fatimarashid15.github.io/Portfolio/**

## 🎨 Design & Features
- **Clean Typography:** Built using Google Fonts (*Playfair Display* for headings and *DM Sans* for body text).
- **Responsive Layout:** Designed to look great on desktop, tablets, and mobile screens.
- **Modern UI:** Styled with a premium dark navy, gold, and cream color palette.
- **Sections Included:** About Me, Skills, Projects, and Contact.

## 🛠️ Tech Stack Used
- **HTML5:** For semantic and structured web layout.
- **CSS3:** For custom styling, layout structures, and smooth hover animations.

## 📂 How to Run Locally
1. Clone or download this repository.
2. Open the `index.html` file in any modern web browser.
